项目简介
智眼启明是面向视障人群打造的边缘 AI 智能盲杖，基于 K230 AI 芯片、树莓派 5、天问 51 硬件平台，结合轻量化 YOLOv8 算法与 ECA-Net 注意力机制，解决传统盲杖仅物理探障、无法提前预警、难以应对复杂路况、盲道被占用等痛点。项目集障碍物识别、语音预警、跌倒检测、北斗 / GPS 定位、Web 安全监护、SOS 紧急呼救于一体，为千万视障人士提供全天候智能出行保障。
本项目遵循 MIT 协议 开源，欢迎各位开发者、嵌入式与 AI 爱好者参与共建、二次开发与功能拓展。
项目背景
据中国残联数据，截至 2023 年我国视力障碍人士超 1700 万，出行辅助需求十分迫切。结合国家 “十五五” 无障碍建设、残疾人保障相关规划，针对传统盲杖三大缺陷：无提前预警、仅依赖物理探障、无法适配复杂路况，本项目采用端侧 AI 技术，实现多类别障碍物实时识别与语音预警，全方位提升出行安全性。
核心功能
🚧 多类障碍物识别：支持 12 类出行障碍物检测，模型置信度＞0.6，识别准确率＞0.7
🔊 实时语音预警：边缘端低延迟推理，本地语音播报路况风险
🚨 跌倒检测报警：搭载 MPU6050 姿态传感器，摔倒自动触发告警
📍 定位与导航：北斗 + GPS 双模定位，对接百度地图 API，支持轨迹记录与路线导航
💻 Web 安全监护平台：设备状态查看、实时定位、预警推送、紧急联系人、SOS 一键呼救
🌤️ 全天候运行：算法与硬件适配日间、夜间等多种出行场景

技术架构
图像采集：GC2093 摄像头实时采集路面画面
AI 模型：轻量化 YOLOv8 + ECA-Net 注意力机制，优化特征提取与识别精度
端侧协同：K230 识别路况 → 树莓派 5 决策处理 → 天问 51 执行语音播报，形成感知 - 决策 - 执行闭环
上位机平台：Python 开发 Web 监护系统，实现数据可视化、远程监护与紧急呼叫
数据集与模型指标
训练数据集：近 6000 张实景标注图片，覆盖道路、盲道、人行道等场景
识别类别：12 类常见出行障碍物
模型效果：置信度＞0.6，识别准确率＞0.7，可流畅运行于低算力边缘设备
